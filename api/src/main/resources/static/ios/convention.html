我们致力于通过在线视频直播为全球用户提供一个实时发现和创建各种新奇的交互内容的平台。为了使得在我们平台上进行的直播活动始终是健康安全的，我们制定了一系列社区公约，目的是尽可能地保证用户的良好体验。我们尽最大的努力保持使用者的健康和安全，但是受限于各种客观条件，我们无法完全保证。如果用户违反下面的这些条款和规则，可能会导致我们暂停为您提供服务甚至永久禁用。
同时请注意，在适用的情况下，我们有权向当地公安机关举报有犯罪嫌疑的活动。
1、禁止暴力和/或自残行为
安全是我们的最高优先级。我们非常严肃地对待暴力威胁和自残行为。如果我们察觉到对公共安全的破坏风险或者直接威胁，我们可能会删除违反这些规定的内容，并且将相关内容向执法机构报告。同样，我们坚决禁止对他人的威胁，如若发生，我们可能会永久禁止您使用服务。
2、坚决禁止裸露或者露骨言行
无论男女，在镜头前必须始终保证衣着端庄。如果在衣衫不整的情况下在镜头前直播将面临被暂停或者永久禁止使用服务。 直播用户禁止暴露着装所覆盖的皮肤。禁止以任何身份鼓励裸露行为，如果发现，将会暂停或者永久禁止该用户使用服务。女性直播用户必须始终保持胸部不能暴露，违反规定的将会面临暂停或者永久禁止使用服务。
3、禁止欺诈，骚扰或者带有仇恨的言论
用户必须尊重社区内的其它成员。不容许嘲弄，欺诈，仇视，种族歧视或者骚扰社区内的其它用户。虽然我们尊重言论自由，但是绝不允许基于种族，国籍，宗教，性，性别，性取向，残疾或者其它疾病进行人身攻击。
4、禁止在屏幕前使用违禁物品
在屏幕前使用违禁药物，药物器具或者吸烟都是禁止的，在屏幕前喝酒也是禁止的。
5、禁止针对应急部门或者热线进行恶作剧
我们绝不允许用户对应急部门或者紧急热线进行恶作剧，和上述行为相关的任何用户都将被永久禁止使用服务。﻿
