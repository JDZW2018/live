傻小白到老油条的转变过程
在开始正式介绍经验之前，先做个简短的自我介绍，我叫罗帅卿，来自上海大同中学PGA国际课程班，我曾参加新东方的暑期ACT培训，成绩提升显著，并在九月份的ACT考试中获得32分的好成绩，在此非常感谢新东方老师们的辛勤付出，我也为能用优异的成绩回报老师而自豪。回忆起在新东方培训的日子，画面历历在目，原来我不是一个人在奋斗，而是一个强大的团队在战斗！ 我每天下午去新东方持续学习到晚上离开，连续两周的高强度学习使我对于整个ACT考试有了较为深入的了解。在老师的悉心指导下，我也掌握了不同类型题目的基本解题方法。每个老师都各具特色、教学风格迥异，下面就让我来为大家详细讲述我在新东方学习的心得。 教我ACT语法的那个会占卜八卦和星座的老师让我印象颇为深刻。她讲课幽默风趣，让枯燥乏味的数个小时充满欢乐活泼的气氛，大家在她的课上也变得积极活跃。同时，她也将考试题型做了详细的分类，强调对句子的分析与理解。每种经典题型她都会给出我们详细的解题步骤并强调需要注意的地方。此外，在学习完一种新的题型后，她会立即让我们训练相同类型的题目，从而加深我们对该知识点的掌握程度，并以此查漏补缺。当我们有任何问题时，她会不断鼓励我们发问，然后再耐心详细地一一回答。如果问的是一道比较经典的易错题，她还会把题目给全班同学再讲一遍，加深记忆过程。在她的精心辅导下，我在课程结束的模考时语法便获得29分的不错成绩，而此次考试的34分高分更是离不开她的功劳。 ACT阅读是孟老师授课的。由于她与ACT语法老师关系甚好，便也多少会一些占卜方面的知识。她经常讲一些ACT语法老师的小故事，让课堂的气氛轻松活跃。在孟老师的指导下，原本复杂的ACT阅读也不再是难题。是她的悉心教诲让我们熟悉了解题步骤，而她针对每种题型做的集中性训练，也让我的对于关键词的定位能力在这种高强度的训练下得到了长足的提升。 ACT数学和ACT科学由同一个老师授课。ACT数学方面，她为我们整理了所有考试中可能出现的数学定理，并针对我们并不是非常熟悉的定理做了详细的讲解。而在科学方面，她用自己精心编著的辅导书为我们对每种题型做逐一的讲解。渐渐的，我发现科学只要做好看题、定位、选答案，就被轻松拿下。在她的引导下，难题也简化了很多。 最后，写作老师为我们讲解了ACT写作的主要步骤，告诉我们如何组织逻辑并整理了好词佳句供我们参考。此外，她督促我们不断地进行实战训练，经常给我们一个题目让我们练习写outline，让我们把每篇写作的逻辑理清楚。同时，她还会认真点评每篇outline。在她的指导下，我渐渐将写托福作文的能力转移到ACT作文上，让自己ACT写作更加条理清晰，层次分明，写起来得心应手。 总而言之，在新东方ACT课程中获得的知识为我日后的英语学习打下了坚实的基础。在此，感谢各位老师以及新东方这个舞台对于我的帮助。现在便是我们收获战果的时候了，让我们愉快地庆祝一下，并预祝依然在备战的同学们取得好成绩，加油！你不是一个人在战斗！
