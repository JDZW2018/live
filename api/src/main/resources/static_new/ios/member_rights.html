一、会员等级规则
1）升级成功后，会员级别会永久保留吗？
会员的有效期是自升级后一年，级别过期后，我们会根据您过去一年的成长值，重新调整您的会员级别。
2）会员级别的有效期限能延长吗？
当会员级别到期后，如未能达到升级标准，但凡会员到期前12个月消费金额大于0元，则当前会员级别就可以顺延一年。
3）我是钻石会员，会员级别到期后没达到延长升级条件，会如何降级？
会员级别到期后，如果未能达到升级及延长有效期条件，将顺次下调一个会员级别（但最低只降到一星会员），同时按照下调日更新会员级别有效期。
二、等级特权详细说明
1）会员升级
权益有效时间：在升级后一周内；
权益内容：您将额外获取100个成长值和100个牛大头，并获得对应等级所享有的权益；
2）节日特权
权益内容：以节日内容为准；
权益对象：教头学院一星及以上会员
3）投诉绿色通道
权益内容：归来后若有投诉均有资深坐席受理，并优先响应，优先解决；
权益对象：教头学院三星及以上会员。
